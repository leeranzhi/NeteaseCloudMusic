# NeteaseCloudMusic
网易云音乐小工具demo

- 依据网易云音乐用户分享歌曲的链接，得出用户名称及个人主页信息
- 查看个人主页信息，高仿网易云音乐个人主页界面

1.跳转到本应用

> 跳转方式支持：

- 网易云音乐-->分享--->选择应用
- 微信/QQ-->从浏览器打开--->选择应用
